Navigator is my personal offline AI command console.

## Current Features

- Local AI using Ollama
- Supports:
  - Llama
  - DeepSeek
  - Gemma
  - Qwen
- Mission logging
- Git version control

Build a modular AI assistant that works locally and helps with:

- Planning
- Journaling
- Bible study
- File management
- Programming
- Research
